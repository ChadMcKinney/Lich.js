combL

A comb delay line with linear interpolation.

Syntax

combL

  • maxDel
  • del
  • decay

Arguments:

  • maxDel

    Max delay time in seconds

  • del

    Delay time in seconds

  • decay

    Time for the echoes to decay by 60 decibels.

Example:

let test del => impulse 1 >> combL del del 1 >> out 0

let t = test 0.1

stop t

env2 Class

Defined in: Soliton.js/SuperCollider.js:1117
Module: Envelopes
Parent Module: Lich.sc UGens

Envelope generator. Used for amplitude envelopes, will NOT automatically free the synth when finished.

Constructor

env2

(
  • levels
  • times
  • shape
  • input
)

Defined in Soliton.js/SuperCollider.js:1117

Parameters:

  • levels Object

    The levels that the envelope will move through

  • times Object

    The times it takes for the env to move between levels. Should be 1 item less than levels

  • shape Object

    Either a shape number or string. Some examples: -4, 0, 1, "linear", "squared"

  • input Object

    The input will be scaled according to the envelope. This can be a ugen or number.
